# ToDo CLI App (Python)

Aplicación de consola desarrollada en Python para gestión de tareas.
Permite agregar, editar, eliminar, completar, buscar y filtrar tareas.
Las tareas se almacenan de forma persistente usando un archivo JSON.

## Funcionalidades
- CRUD de tareas
- Búsqueda por texto
- Filtro por estado
- Persistencia con JSON
- Validaciones de entrada

## Tecnologías
- Python 3
- JSON
- Programación modular

## Ejecución
```bash
python main.py
